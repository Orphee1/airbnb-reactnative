## Install the project

Clone this repository :

```
git clone https://github.com/Orphee1/airbnb-reactnative.git
cd airbnb-reactnative

```

Install packages

```
npm install
```

When installation is complete, run it:

```
npm start
```

## Technologies

- React - version 16.9
- React Native
- React Navigation
- Expo - version 36.0
- Axios - version 0.19
